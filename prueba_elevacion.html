🏔️ Prueba de servicio de elevación

Esta página comprueba si tu navegador puede consultar el servicio gratuito Open Topo Data desde una página web. Si funciona, podremos integrarlo en la app.

Cómo interpretar:
• Verde (✅): el servicio respondió y el navegador lo permitió. Podemos integrarlo.
• Rojo con "CORS" o "Failed to fetch": el navegador bloqueó la consulta. Habría que probar otro servicio o un plan B.
• Rojo con otro mensaje: el servicio respondió pero con error (ej. coordenada fuera de rango o servicio caído).

Coordenadas de ejemplo: Tunja (5.5353, -73.3678). Puedes cambiarlas por las de un lote real para ver si da una altura coherente con la zona.
